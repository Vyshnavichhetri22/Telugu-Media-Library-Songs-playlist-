# Telugu Media Library

This is a simple Telugu Media Library built using only HTML.

## Features
- Movie-based song collection
- Clickable images to open song pages
- Audio playback using <audio> tag
- Multiple HTML pages

## Technologies Used
- HTML only

## Purpose
This project is created for practicing basic HTML concepts like links, images, and audio.
